# 最终输出模板

完成评估后，严格按照以下模板输出最终报告，不得增加或减少任何输出模块。

---

## 标题：【客户名称】 - 新客户预审报告

**【目标企业最终评级】：[X 级 - 对应标签名称]**

**核心风控画像总结：**
基于该企业信用评估的最终结果，用不超过200字的精炼专业语言总结其核心风险特征、信用亮点及关键关注点。禁止输出任何评级规则、红线说明或评级调整过程。

## 模块一：客户基本信息

1. **工商登记信息**
   - 统一社会信用代码
   - 法定代表人
   - 成立日期
   - 注销/吊销日期
   - 注册资本
   - 实缴资本
   - 注册地址
   - 企业类型
   - 经营（业务）范围

2. **股权结构**
   - 主要股东及持股比例
   - 实际控制人、实际控制人背景、对外投资情况

3. **核心高管信息**
   - 法定代表人、财务负责人及主要管理人员姓名、任职情况
   - 高管对外投资情况

4. **工商及人员变更**
   - 近三年工商登记、实控人、股东、法人、财务负责人、高管变更情况

5. **企业资质**
   - 经营许可证名称、高新技术企业证书名称、创新型中小企业认证、专精特新/小巨人认证、知识产权情况

6. **涉税信息**
   - 纳税信用等级、近三年纳税情况、有无欠税、税务处罚等异常情况

7. **合规与风险信息**（信息数量较多时，模型整合总结后展示）
   - **行政处罚信息**：近三年工商/税务/行业监管/环保/消防等处罚内容、金额、整改情况
   - **司法处罚信息**：近三年诉讼/仲裁案件（数量、类型）、被执行记录、失信被执行、限制高消费
   - **舆情信息**

8. **经营信息**
   - 经营情况、经营异常名录、科技型企业认定、企业年报社保信息、海关信用等级

## 模块二：企业供应链信息

将供应链分析工具的输出结果，结合用户提供的相关有效信息，凝练为一段话输出。

## 模块三：企业产业链信息

1. 将产行业风险分析 Skill 的输出结果凝练为一段话输出，至少应包括其产业链趋势分析、产业分级结果。
2. 【产业分级】：[成熟产业/高增长前景产业/未来发展产业]

## 模块四：企业财务信息

- 企业存在有效财务数据，已执行财务分析后，直接引用财务分析 Skill 输出中的“4.2 财务综合评价”结论段，原样展示，无需再加工，不设置小标题。
- 企业不存在有效财务数据，输出“无有效财务数据，未执行财务分析”
