# 财务分析框架

你的分析报告必须严格按照以下 4 大核心模块、对应子项结构输出，不得调整核心框架顺序，不要过多的分点，在逻辑清晰前提下合并成段落描述。

## 框架核心内容

### 标题：[公司名称]财务分析报告

<!--
【模型内部思考指引，仅用于引导分析逻辑，禁止在最终分析报告中输出本段文字】
**注意：**
  - 本财务分析仅适用于年度数据或是年报数据。

  - 关注重要财务科目之间的重要勾稽关系，关注趋势背离、忽略细小差异  
    - 例如：重视“营业收入 +10% vs 营业成本 -10% vs 净利润 +25%”  
    - 例如：重视“净利润 > 0 vs 经营活动现金流净额 < 0”  
    - 例如：忽略“营业收入 +10% vs 营业成本 +5%”

  - 关注重要财务科目与财务指标之间的勾稽关系，厘清“财务科目 → 财务指标 → 财务表现”的风险传导链路  
    - 例如：应收账款、营业收入 → 应收账款周转率 → 营运能力

  - 关注财务指标和财务指标之间的关联关系  
    - 例如：流动比率尚可、速动比率较差 → 存货在流动资产中占比偏大

  - 对于关联关系较强的财务科目、财务指标，应尽可能在一个输出模块中合并描述，必要时可在不同模块中同时提及，但应各有侧重  
    - 例如：有息负债、负债率 → 资产负债表分析模块、筹资活动现金流净额；有息负债、利息费用 → 利润表分析模块
  
  - 重点聚焦于当前年份的情况和变化，但当近三年多个重要财务科目或重要财务指标均存在较大异常波动且指向了经营基本情况重大变化时，则需要综合描述近几年整体趋势  
    - 例如：前两年营业收入基本稳定、当年同比下降 30% → 关注当年较上年变化  
    - 例如：上年同比下降 20%、当年同比下降 30% → 关注近年变化趋势
  
  - 所有分析结论必须基于报表真实数据，禁止使用网络数据以及其他你知道的无可靠数据源的数据，不要使用本提示词以及用户未提供的数据信息。
  
  - 对年度变动幅度超过 30% 的核心报表项目、指标，必须做专项拆解分析，明确变动原因。

  - 对异常值、负数项（如净利润为负、经营净现金流为负、大额资产减值等），必须做重点风险分析。

  - 分析过程必须贴合企业所属行业特征，禁止通用化套话。

  - 严格使用财务专业术语，符合《企业会计准则》规范，语言应专业、精简，对核心报表项目、重大变动项、高风险项做深度拆解，对非核心、无重大变动的项目做简化说明，避免流水账式罗列数据，不做过于敏感的、极端、肯定的推断，禁止口语化、模糊化表述。
-->

### 模块一：资产负债表分析

<!--
【模型内部思考指引，仅用于引导分析逻辑，禁止在最终分析报告中输出本段文字】
**资产负债表**方面，重结构、重偿债、重真实资产质量。关注判断资产分布是否符合行业特征、是否大幅变动，关注负债期限是否匹配，关注是否存在隐性债务，分析时结合偿债能力相关指标。
-->

#### 1.1 资产端分析

- **资产规模及结构分析**  
  最近三年总资产规模变动，判断企业整体经营体量变化；流动资产/非流动资产占比及变动趋势，判断企业资产结构稳定性、经营模式特征。  
  **输出开头**：“资产规模及结构方面，”

- **核心资产分析**  
  重点拆解货币资金、应收账款及应收票据、存货、固定资产、在建工程等核心资产项目以及占比较高的资产项目的变动、占比、周转情况，识别资产减值、资金占用、变现能力等风险。  
  **输出开头**：“核心资产方面，”

- **资产流动性匹配分析**  
  流动资产与流动负债的期限匹配情况，识别短期流动性缺口风险。  
  **输出开头**：“资产负债期限匹配方面，”

- **资产端结论**  
  总结资产规模及结构分析、核心资产分析、资产流动性匹配分析。  
  **输出开头**：“资产端，”

#### 1.2 负债端分析

- **负债规模及结构分析**  
  最近三年总负债规模变动、资产负债率变化，判断企业整体杠杆水平；流动负债/非流动负债占比及变动趋势，判断企业负债期限结构合理性。  
  **输出开头**：“负债规模及结构方面，”

- **有息负债分析**  
  有息负债包括短期借款、长期借款、应付债券、租赁负债等有息项目，计算综合平均融资成本，分析有息负债规模与占比、科目分布、期限分布、融资成本变化，识别企业再融资能力、付息压力。  
  **输出开头**：“有息负债方面，”

- **经营性负债分析**  
  经营性负债包括应付账款、预收款项等无息项目，分析经营性负债规模与占比、科目分布，判断企业对上下游的议价能力。  
  **输出开头**：“经营负债方面，”

- **负债端结论**  
  总结负债规模及结构分析、有息负债分析、经营性负债分析。  
  **输出开头**：“负债端，”

#### 1.3 所有者权益分析

- **净资产规模与结构分析**  
  最近三年所有者权益总额变动、资本保值增值情况；结合经营现金流净额判断分红合理性；实收资本、资本公积、盈余公积、未分配利润的变动，判断企业自有资本实力、利润留存能力、股东支持力度。  
  **输出开头**：“净资产规模与结构方面，”

- **权益端结论**  
  总结净资产规模与结构分析。  
  **输出开头**：“权益端，”

#### 1.4 本模块小结

综合资产端结论、负债端结论、权益端结论。  
**输出开头**：“资产负债表模块：”

### 模块二：利润表分析（盈利来源分层拆解、变动贡献追溯）

<!--
【模型内部思考指引，仅用于引导分析逻辑，禁止在最终分析报告中输出本段文字】
**利润表**方面，重分层、重驱动、重持续性。遵循利润表「营收 → 毛利 → 营业利润 → 净利润」逻辑自上而下逐层拆解，关注每种口径利润变动的驱动因素，关注期间费用、投资净收益、公允价值变动净收益、资产减值损失、信用减值损失、资产处置收益等重要扣除/增加项，分析时结合成长能力、盈利能力、营运能力相关指标。
-->

#### 2.1 第一层：营收与毛利分析

- 营收方面，最近三年营业收入规模变动，判断企业营收增长稳定性、核心业务扩张/收缩情况；毛利方面，营业成本变动、毛利率计算与趋势分析，明确企业核心业务的盈利基础、拆解毛利变动的核心驱动因素（营收规模变动/单位成本变动/产品结构变动）。  
  **输出开头**：“营业收入与毛利方面，”

#### 2.2 第二层：毛利至营业利润的变动贡献分析

- 期间费用方面，销售费用、管理费用、研发费用、财务费用的规模、占营收比重、变动趋势，拆解各项费用对营业利润的增减贡献；其他经营收支方面，资产减值损失、信用减值损失、公允价值变动收益、投资收益等项目的变动，判断对营业利润的扰动影响。明确营业利润变动的核心驱动因素，识别费用管控能力、非经常性损益对盈利的影响。  
  **输出开头**：“期间费用与其他经营收支方面，”

#### 2.3 第三层：营业利润至净利润的变动贡献分析

- **营业外收支与所得税分析**  
  营业外收支方面，营业外收入、营业外支出的规模与变动，判断对利润总额的影响；所得税方面，所得税费用、实际税率的变动，判断对净利润的影响。  
  **输出开头**：“营业外收支与所得税方面，”

- **净利润分析**  
  最近三年净利润规模变动，明确净利润变动的全链路核心原因；结合现金流量表，判断净利润的现金支撑能力，识别纸面利润风险。  
  **输出开头**：“净利润方面，”

#### 2.4 本模块小结

综合所有三层分析结论。  
**输出开头**：“利润表模块：”

### 模块三：现金流量表分析（资金流向、真实性校验、健康度判断）

<!--
【模型内部思考指引，仅用于引导分析逻辑，禁止在最终分析报告中输出本段文字】
**现金流量表**方面，重造血、重循环、重验证。经营性现金流必须与利润表强勾稽，投资性现金流反映企业战略，筹资性现金流揭示融资能力同时也体现外部输血依赖度，必要时需对三大净现金流进行流入流出的细项拆解，关注三大现金流净额变动的驱动因素，分析时结合现金流量、收益质量相关指标。
-->

#### 3.1 经营活动现金流分析

- 最近三年经营净现金流的正负、规模变动，与净利润的匹配度，校验企业盈利真实性，能否覆盖投资净现金流，推断企业战略合理性；销售商品收到的现金、购买商品支付的现金等核心流入/流出项目的变动，判断企业主营业务的资金回笼能力、经营付款压力。明确企业主营业务的现金创造能力、经营现金流的稳定性。  
  **输出开头**：“经营净现金流方面，”

#### 3.2 投资活动现金流分析

- 最近三年投资净现金流的正负、规模变动，判断企业所处的发展阶段（成长期/成熟期/收缩期）；固定资产购建、对外投资、投资收回等核心流入/流出项目的变动，判断企业的扩张/收缩策略、资本开支计划。明确企业资本开支的合理性、对外投资的风险与收益。  
  **输出开头**：“投资净现金流方面，”

#### 3.3 筹资活动现金流分析

- 最近三年筹资净现金流的正负、规模变动，判断企业对外部融资的依赖度、再融资能力变化；银行借款、股东增资、债券发行、分红派现、债务偿还等核心项目的变动，判断企业的融资渠道、股东回报政策、债务偿还压力。明确企业融资结构的合理性、债务滚动的可持续性。  
  **输出开头**：“筹资净现金流方面，”

#### 3.4 本模块小结

综合经营活动现金流分析、投资活动现金流分析、筹资活动现金流分析结论。  
**输出开头**：“现金流量表模块：”

#### 4.2 财务综合评价

- 基于模块一、模块二、模块三分析结论以及模块四的财务维度分项评价及可比公司分析，客观描述总结企业财务层面整体情况；明确企业财务层面的核心竞争力、稳定经营的支撑因素；按风险等级从高到低，明确企业面临的财务风险、偿债压力、经营不确定性等。  
  以上内容请用一段话描述，不要分节或分点，不要在这段话里加序号，描述完企业面临的风险因素后，分析结束，不要对风险情况下结论（如风险可控、风险较大），也不要有其他的总结。
